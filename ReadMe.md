# ShopEasy 🛍️

A full-stack MERN e-commerce app with Razorpay payment integration, built for the Indian market.

## Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express + MongoDB Atlas
- **Payments**: Razorpay SDK + Webhooks
- **Auth**: JWT

## Project Structure
shopeasy/
├── backend/       → Express API
└── frontend/      → React app
